<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time AI suggestions that analyze audience reactions during live performances and adapt visuals on the fly.
- Introduced 15 new monetization avenues, including B2B partnerships and festival licensing for asset creators.

## Improvements

- Enhanced storyboard creator with drag-and-drop setlist integration for faster video prototyping.
- Performance boost in AI-driven asset creation, reducing generation time by `50%`.

## Bug Fixes

- Fixed sync issues between audio-reactive VJ loops and live inputs for DJs and musicians.
- Resolved marketplace upload errors for custom templates and AI styles.

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched asset marketplace with user-generated templates, AI styles, and monetization tools.
- New setlist organizer featuring intuitive planning for VJs, YouTubers, and filmmakers.

## Breaking Changes

- Updated API endpoints for asset generation; migrate to `https://api.vfxstudios.com/v2/assets`.
- Required authentication update: Use `{API_KEY}` with Bearer token format.

## Improvements

- Uptime SLA improved to `99.9%` with better cloud infrastructure.

</Update>

<CodeGroup tabs="Dashboard,API">
  ```javascript
  // Check version via API
  const response = await fetch('https://api.vfxstudios.com/v2/version', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });

  ```
  ```python
  import requests
  headers = {'Authorization': f'Bearer {YOUR_API_KEY}'}
  response = requests.get('https://api.vfxstudios.com/v2/version', headers=headers)

  ```
